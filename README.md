# Bancos Brasileiros
Listagem de bancos comerciais brasileiros

A listagem contém 250 bancos com as sequintes informações:

 - Código do banco - FEBRABAN / BCB
 - Razão Social
 - CNPJ
 
 Os formatos disponíveis são: 
 - **T-SQL**;
 - **JSON**
